The Great Suspender :: Settings








Suspended Tab History

A history of the tabs that you have suspended in the past is recorded. In the (unlikely :) event that The Great Suspender should ever crash, you can recover lost tabs from the history page.

Notes about Screen Capturing:

Sorry to all that were using this feature but I have decided to remove the screen capture functionality as it was causing the extension to consume too much memory. If there is enough demand for it, I can reintroduce this functionality and offer it as a seperate extension.
