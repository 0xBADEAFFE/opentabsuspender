The Great Suspender :: New Version is Out!

Thank you for trying out The Great Suspender. I have been working hard over the last few days to iron out some bugs that have been raised. Most notably the buttons not appearing in the drop down menu. But also some quite significant background issues.

Because of the way this extension works, it is important to note that every time I release an update, all of the tabs that are currently suspended in your browser will disappear as the extension restarts. I have been working on a restore tab much like this screen that should help you easily recover after an update. From this screen you can easily restore all your suspended tabs, either in a suspended state or back to their original state.

Please Note: Due to a bug in the previous version, you might find that, just this once, there are a lot of tabs listed here that are from previous sessions. My apologies for this. Until you receive this new update I am unable to tell the difference between a tab that has been suspended and is still open, and a tab that has been suspended and then closed. This has been fixed in this latest release and the session recovery page seems to work very well now.

Thanks again for using my extension. Any feedback / suggestions are always welcome. You can email me directly at deanoemcke@gmail.com or use chromes various feedback mechanisms.
