Important note about new version of The Great Suspender:


It has come to my attention that when upgrading to a new version of this extension, all you currently suspended tabs will dissappear. I'm very sorry about this!


In this new version I have incorporated some crash recovery support so that if the extension crashes, or if you reload or update it, it will automatically restore any tabs that were currently suspended.


However, that does not help you right now if you have just upgraded from an older version. In this case I suggest you try going to the history tab. You should be able to recover any lost tabs from there. Note: You will not need to do this in future now that you have the new version with crash recovery :)
