The Great Suspender - New Version is Out!

You've just updated to a shiny new version of The Great Suspender. Before I get into what's new, I would like to apologise for any suspended tabs that may have just disappeared from your browser. It is an unfortunate side-effect of installing this new release. However, these tabs should be able to be recovered from the history page. And the good news is that this new release has fixed the issue meaning upgrades from now on in should be smooth sailing!
This new version comes with a lot of new features and some much requested bug fixes. Sorry they've taken so long.

Bug Fixes:

- Fixed a bug around whitelisting not working correctly. - Fixed an issue with the extension not obeying the requested suspension timeouts.

New Features:

- Full support for chrome syncing! This means that all your options and whitelisted tabs will be synced across all your logged in chrome browsers. You will also be able to load suspended tabs from your synced history.

Useability Improvements:

- Chrome sync support. Suspended tabs will now appear normally over chrome sync. - Improved error recovery. You will not lose your tabs if the extension restarts or is uninstalled. - Introduced an option to never suspend pinned tabs. - New 1 minute and 5 minute options for time to auto-suspend. - Ability to choose image quality of screen preview. - Can now prevent tabs with a half-completed form from suspending and losing your entered text. - New option to force a cache refresh when reloading a tab. - New recovery screen that more accurately restores previous sessions as well as a configurable number of sessions to save. -

Thanks again for using my extension. Any feedback / suggestions are always welcome. You can email me directly at deanoemcke@gmail.com or report any issues on my github page.
